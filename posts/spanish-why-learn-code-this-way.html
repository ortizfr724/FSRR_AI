¿Por qué aprender a programar de esta manera?

Bienvenidos a FSRR-AI: Bases Fundamentales para Escalar a Full Stack Rails: IA: Aprende el Prompt, Acorta la Brecha. Me embarco en un proyecto para construir un blog estático, y mi enfoque es un poco diferente. En lugar de lo que algunos podrían llamar "programación por vibras", un término a menudo asociado con Andrej Karpathy donde simplemente podrías indicarle algo a una IA y copiar y pegar los resultados, mi objetivo es una inmersión más profunda.

Este blog documentará mi viaje para comprender los bloques de construcción fundamentales del desarrollo web: HTML, CSS y JavaScript, guiado por el plan de estudios integral de The Odin Project. Pero hay un elemento clave en mi proceso de aprendizaje: estoy utilizando la IA como mi asistente y tutor constante. Esta estrategia intencional está diseñada para mostrar a otros principiantes cómo este método puede ser increíblemente beneficioso. Al comenzar con un sitio web estático básico y escribir activamente el código, no solo estoy aprendiendo sintaxis; me esfuerzo por comprender el significado y el propósito subyacente de cada concepto, a menudo a través de la exploración guiada con la IA.

Estos son algunos de los beneficios principales de este enfoque de aprendizaje intencional:

  • Escritura Activa de Código: Escribir código a mano, en lugar de copiar pasivamente, refuerza la comprensión y solidifica los conceptos.
  • Comprensión Conceptual Más Profunda: Interactuar con la IA me permite preguntar "¿por qué?" y profundizar en el razonamiento detrás de diferentes métodos de codificación.
  • Guía y Retroalimentación Personalizadas: La IA proporciona explicaciones y comentarios a medida, lo que me ayuda a alinearme con las mejores prácticas desde el principio.
  • Desarrollo Gradual de Habilidades: Construir a partir de un sitio web estático permite una comprensión paso a paso de las tecnologías centrales antes de pasar a marcos más complejos como Ruby on Rails.

En última instancia, mi objetivo con FSRR-AI es proporcionar una experiencia única y motivadora para los principiantes. Quiero compartir la emoción de desmitificar el desarrollo web, demostrando que con un enfoque reflexivo y el apoyo de la IA como asistente educativo, cualquiera puede construir una base sólida y comenzar con confianza su viaje de programación.

Este viaje evolucionará a medida que crezca el blog. Verán mejoras en la estructura HTML utilizando clases de utilidad, la aplicación de posicionamiento y espaciado, y la introducción de CSS Orientado a Objetos. Progresaremos gradualmente a través de las propiedades de CSS, técnicas de diseño como Flexbox y Grid, todo mientras mantenemos la vista puesta en el objetivo final: comprender el desarrollo Full Stack a través de la lente de The Odin Project. Esto les permitirá aprender junto con el blog, presenciando la progresión y comprendiendo el "por qué" detrás de cada paso.

Durante los últimos seis meses, mientras he navegado por The Odin Project, la IA ha sido fundamental para aclarar temas complejos y descubrir conocimientos más profundos sobre los aspectos técnicos del desarrollo web full-stack. Algunas de las áreas que he explorado con la IA incluyen:

  • Comprender las mejores prácticas para usar etiquetas HTML semánticas para dar significado y estructura al contenido web. Es importante tener en cuenta que, si bien HTML se centra en el significado semántico, CSS se utiliza para dar estilo visual a estos elementos mediante selectores.
  • Buscar recomendaciones de libros de HTML y CSS para principiantes que prioricen explicaciones claras y ejemplos prácticos.
  • Explorar los principios de diseño básicos y las mejores prácticas para aplicarlos a proyectos para principiantes, al mismo tiempo que se comprenden los errores comunes que se deben evitar.

Para ilustrar esto, estos son algunos de los componentes de configuración inicial dentro de The Odin Project que exploré con la guía de la IA:

  1. Comprender la configuración del entorno de trabajo, incluidas las máquinas virtuales y los sistemas operativos duales, y las consideraciones para diferentes sistemas operativos.
  2. Aprender los fundamentos de la línea de comandos.
  3. Configurar GitHub, incluida la creación de claves SSH y la comprensión de los repositorios.
  4. Familiarizarse con VSCode, el editor de código recomendado.

Este blog, FSRR-AI, se basa en la creencia de que al comprender las bases fundamentales con la ayuda de la IA (al aprender el prompt y acortar la brecha entre la comprensión humana y la inteligencia artificial), los principiantes pueden escalar con confianza sus habilidades hacia el ámbito del desarrollo Full Stack Rails y prosperar en el panorama cambiante de los equipos de desarrollo aumentados por la IA. ¡Únanse a mí en este viaje mientras desempacamos juntos el mundo del código!

¡Excelente! Ahora que hemos explorado la filosofía detrás de aprender a programar de esta manera, ¿qué sigue?
